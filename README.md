# Olist E-Commerce Sales and Customer Analysis

## Project Overview
This project analyzes the Olist E-Commerce dataset using Power BI. The dashboard provides insights into sales performance, customer behavior, payment methods, order status, and product categories.

## Objectives
- Analyze total revenue and orders
- Identify customer review patterns
- Analyze payment methods
- Compare revenue by customer state
- Identify top-performing product categories

## Tools Used
- Power BI
- Microsoft Excel
- Olist E-Commerce Dataset

## Dashboard KPIs
- Total Revenue: 16.01M
- Total Customers: 99K
- Total Orders: 96K
- Average Review Score: 3.84

## Dashboard Visualizations
- Revenue by Year
- Review Score Distribution
- Revenue by Customer State
- Order Status
- Payment Value by Payment Type
- Top Revenue Product Categories

## Key Insights
1. Revenue increased consistently from 2016 to 2018.
2. More than half of customers gave 5-star ratings.
3. Credit card is the most preferred payment method.
4. Most orders were successfully delivered.
5. A few customer states generate the highest revenue.
